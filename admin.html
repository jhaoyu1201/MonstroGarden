🐾 物種管理

⚔️ 戰鬥參數
所有戰鬥計算的全域參數。擊退值現在是各物種獨立設定（在物種管理裡調整）。
🛒 道具管理
可以新增、編輯、刪除商店道具。
effect 識別碼：feed(餵食) · heal(回復) · hatch(孵蛋) · shiny_boost(色違提升) · custom(自訂，需在遊戲代碼中實作)
🎵 音樂設定
所有音樂檔案請放至 assets/music/ 資料夾。填寫檔名即可（含副檔名）。
留空 = 使用程式生成的音樂（飼養箱/戰鬥場景有預設生成音樂；其他場景靜音）。
🔊 音效設定
所有音效檔案請放至 assets/sfx/ 資料夾。填寫檔名即可。
留空 = 使用程式生成的音效。
🖼️ 場景圖片
背景圖片請放至 assets/scenes/ 資料夾。建議尺寸：1920×1080 或按實際版面比例。
留空 = 使用程式繪製的背景。
🎨 UI 圖片
UI 素材請放至 assets/ui/ 資料夾。留空 = 使用預設 CSS 樣式。
📈 升級經驗表
每個等級所需的累計總經驗值，數值必須遞增。
🎯 技能平衡
調整被動技能數值、狀態效果、系統機率。儲存後在遊戲中即時生效。
📤 匯出 species.js
調整完成後點「生成」，複製內容覆蓋 species.js 檔案，所有設定就會永久保存。
「儲存並套用」直接寫入 localStorage，重新整理遊戲頁即可看到效果。換電腦前請用「匯出 JSON」備份。
